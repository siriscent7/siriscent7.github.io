# siriscent7.github.io

> Personal portfolio of Anushka Siri Raghunandan — live at [siriscent7.github.io](https://siriscent7.github.io/)

---

## About

A minimal, interactive portfolio built with pure HTML, CSS, and Vanilla JS —
no frameworks, no build steps. Designed with sustainability in mind: dark-first,
system fonts, zero dependencies.

Features a floating **terminal emulator** where you can navigate the portfolio
via CLI commands, alongside a clean scrollable main page.

---

## Features

- ⚡ Interactive terminal console — type `help` to explore
- 🌙 Dark / light theme toggle
- 🎞️ Staggered fade-up hero animations
- 📱 Fully responsive
- 🌱 Low-carbon design — dark default, no external fonts or frameworks

---

## Stack

| Layer    | Technology                        |
|----------|-----------------------------------|
| Markup   | HTML5                             |
| Styling  | CSS3 (custom properties, no libs) |
| Logic    | Vanilla JavaScript (ES6+)         |
| Hosting  | GitHub Pages                      |

---

## Terminal Commands

Once on the site, click the terminal and try:
- help → list all commands
- about → who I am
- experience → work history
- projects → things I've built
- skills → tools & technologies
- education → academic background
- resume → open resume PDF
- contact → get in touch
- clear → clear the terminal
